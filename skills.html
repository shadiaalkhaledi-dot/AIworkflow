Custom Modules

The Skills Library

My reflexes, available for adoption. A "skill" is a small instruction file that teaches Claude one specific job — install it once and your AI shows up already knowing how, instead of asking twenty warm-up questions. No prompt-writing. No ritual. Download, install, done.

Quick Setup

How to install a skill file

Four steps, none of them scary. The hardest part is deciding which one to grab first:

  1. Download the skill file

    Click any card's download button below to download the corresponding .skill file to your computer.

  2. Attach it to your chat

    Drag and drop the downloaded file into your active Claude Code or Cowork session, or use the paperclip attachment icon to upload it.

  3. Tell Claude to install it

    Type "install this skill" in the chat. Claude will process the file contents and show a preview card with a Save skill button.

  4. Save and complete

    Click the Save button on the card. The skill is now saved to your shared config file and will be available in all future sessions automatically!

Behind the scenes

What is inside a skill file?

Understanding the structure of a skill file makes it easy to edit it to suit your own needs:

A skill file is a standard ZIP archive (renamed to the .skill extension) containing a single SKILL.md Markdown file. Claude Code and Cowork unpack this archive to register the skill. The markdown file uses YAML frontmatter to declare its triggers, and structured rules to define the agent's task:

File Section Purpose & Customization
YAML Frontmatter (lines 1-4) Defines the skill's name and description. Triggers are automatically derived from the name and description. You can edit this metadata to change the trigger phrase.
Instructions / Markdown Body Contains the precise guidelines, checklists, and agent roles. You can unzip the skill, modify the markdown rules (e.g. adding your own specific spelling preferences or citation formats), and zip it back up to customize it.
Output Format Spec Tells the AI exactly how to output the results (e.g. clean polished text followed by a checklist of flags or reports). Keeping this section clean ensures Claude outputs the results in a predictable way.
Making Your Own Skills

Once you get used to using these skills, you can make your own! Whenever you find yourself typing a repetitive series of prompts in your chat, ask Claude: "Turn what we just did into a reusable skill file named my-skill.skill." Claude will package the instructions and output a downloadable skill file you can install locally or share with others.
