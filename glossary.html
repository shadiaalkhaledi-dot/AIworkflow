Glossary

Shorthand used across this site, in one place, in case you forget what something stood for halfway through.

Apps Script
Google's free, built-in scripting language for Sheets/Docs/Drive. The "logic" layer behind almost every dashboard on this site — reads and writes spreadsheet rows, runs automation, can call out to an AI.
Web App deployment
The Google feature that turns an Apps Script project into a real, shareable URL. Free, no server to maintain — the trade-off is it's built for small-scale personal use, not heavy public traffic.
Obsidian
A note-taking app built on plain text files in a folder ("a vault"), extendable with community plugins. The foundation of the Second Mind system.
Vault
Obsidian's term for a folder of notes it manages. This system uses two: an "Ops" vault (the messy workshop) and a "Main" vault (the trusted library).
Dataview / DataviewJS
A plugin that lets a note run live queries — or, in "JS" mode, actual JavaScript — and render the result inside the note itself. What powers the live home-screen dashboard.
NotebookLM
Google's grounded-research tool — you feed it documents, and it only answers from those documents rather than general knowledge. Used here as a "source desk" for PDF-heavy research like building codes and exam material.
R1 – R4 (Source Reliability Tiers)
A scale from R1 (verified directly against a primary source) down to R4 (unverified — the default for anything NotebookLM produces until it's been checked).
Lane A / Lane B
How claims get classified in the knowledge pipeline. Lane A = tied to a citable rule or code, treated as fact. Lane B = a reasonable interpretation or best practice, which can never override a Lane A claim if the two conflict.
Q-Track / S-Track / F-Track
The three speeds new information can be routed through: Quick (direct answer from existing trusted notes), Standard (a 4-stage check for genuinely new information), and Full (a deep, multi-source investigation used when sources disagree or stakes are high).
The Four Gates (Gate-C / V / E / P)
The checkpoints a piece of information passes through before being trusted: Coverage (did we look broadly enough), Verification (checked against a primary source), Escalation (kicked up for deeper review if unclear), and Promotion (the final move into the trusted vault).
Context Packet
A structured summary of the relevant facts, handed to an "expert" persona before it answers — so every specialist reasons from the same starting information instead of whatever's floating around in the conversation.
Tier 1 / 2 / 3 (Expert Review)
How big the review panel is for a given question: Tier 1 is one specialist, Tier 2 is a small focus group running in parallel, Tier 3 is the full panel plus a final synthesis pass that reconciles everyone's input.
Zone 1 / Zone 2 (Handoff doc)
The two-part structure of a shared collaboration log: Zone 1 is static project context, read once. Zone 2 is an append-only session log — each new session only reads entries since their last visit, not the full history.
PWA (Progressive Web App)
A website that can be "installed" on a phone or computer like a real app, without going through an app store.
